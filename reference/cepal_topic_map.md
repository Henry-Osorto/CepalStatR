# CEPALSTAT Thematic Map (Interactive Tree)

Creates an interactive thematic tree of CEPALSTAT indicators with the
same visual structure as viewer.indicators().

## Usage

``` r
cepal_topic_map(language.en = TRUE, open.browser = FALSE)
```

## Arguments

- language.en:

  Logical. TRUE for English, FALSE for Spanish.

- open.browser:

  Logical. Open in browser (default FALSE).

## Value

HTML widget
